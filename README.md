# Нейросетевая модель для стилизации и реконструкции разметки образовательных математических текстов.


Локальный прототип системы преобразования математической речи и вербализованных математических выражений на русском языке в LaTeX.

Приложение работает по двухэтапной схеме:
1. аудио преобразуется в русский текст с помощью локальной ASR-модели;
2. полученный текст преобразуется в LaTeX локальной seq2seq-моделью ByT5.

После этого результат отображается в виде LaTeX-кода и визуализированной формулы в браузере.

## Возможности

- запись голоса через интерфейс браузера
- загрузка готового аудиофайла
- локальное распознавание речи
- ручная правка промежуточного текста
- генерация LaTeX
- визуализация формулы
- копирование итогового LaTeX
- демонстрационные примеры для быстрого тестирования

## Требования

- Python 3.11
- Git
- Git LFS

## Клонирование репозитория

Сначала необходимо установить Git LFS, затем выполнить клонирование репозитория.

```powershell
git lfs install
git clone https://github.com/kostyabuh/ByT5_forLATEX_diploma.git
cd ByT5_forLATEX_diploma
git lfs pull
```

## Создание виртуального окружения

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

## Установка зависимостей

```powershell
pip install -r requirements.txt
```

## Запуск приложения

```powershell
python app.py
```

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:5000
```

## Как работать с приложением

1. Открыть приложение в браузере.
2. Выбрать один из вариантов:
   - записать голос
   - загрузить аудиофайл
   - подставить демонстрационный пример
3. Получить распознанный текст.
4. При необходимости исправить текст вручную.
5. Запустить генерацию LaTeX.
6. Просмотреть формулу и скопировать итоговый результат.

## Важные замечания

- основная ByT5-модель хранится в репозитории и загружается локально
- при первом запуске модуль распознавания речи может дополнительно скачать локальную ASR-модель в системный кэш
- первый запуск приложения может быть медленнее из-за загрузки и прогрева моделей
- для корректной работы на другой машине необходимо обязательно выполнять `git lfs pull`, иначе веса модели не будут загружены

## Основные файлы

- `app.py` — серверное Flask-приложение
- `asr_local.py` — локальное распознавание речи
- `inference.py` — загрузка и запуск модели генерации LaTeX
- `templates/index.html` — HTML-интерфейс
- `static/app.js` — логика фронтенда
- `static/styles.css` — оформление интерфейса
